# Bus Simulation Model

Имитационная модель движения автобусов по маршруту для оптимизации интенсивности выпуска автобусов.

## Описание

Модель симулирует движение автобусов по линейному маршруту с несколькими остановками. 
Каждая остановка представляет собой систему массового обслуживания с пуассоновским потоком пассажиров.
Цель модели — найти оптимальную интенсивность выпуска автобусов, максимизирующую прибыль.

## Основные характеристики

- Дискретно-событийное моделирование (DES)
- Пуассоновский поток пассажиров на остановках
- Усеченное нормальное распределение интервалов выпуска автобусов
- Сбор метрик: время ожидания, длительность поездки, загрузка автобусов
- Автоматическая оптимизация интенсивности выпуска
