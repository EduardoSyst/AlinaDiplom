# Bus Simulation Model

> Имитационная модель движения автобусов для оптимизации интенсивности выпуска автобусов.

Детальная имитационная модель дискретно-событийного (DES) моделирования движения автобусов по маршруту с несколькими остановками. Модель оптимизирует интенсивность выпуска автобусов, максимизируя прибыль при ограничениях времени ожидания пассажиров.

## Quick Start

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск с примером конфигурации
python main.py --config config/example_uniform.json

# Просмотр всех параметров
python main.py --help
```

## Основные характеристики

- **Дискретно-событийное моделирование (DES)** — точная имитация событий во времени
- **Пуассоновский поток пассажиров** — реалистичное распределение прибытия
- **Усечённое нормальное распределение** — интервалы выпуска автобусов
- **Автоматическая оптимизация** — поиск оптимальной интенсивности для максимизации прибыли
- **Комплексная метрика** — время ожидания, длительность поездки, загрузка автобусов
- **Визуализация результатов** — графики и отчёты в output/plots/
