# Project Roadmap

> Имитационная модель дискретно-событийного моделирования (DES) движения автобусов с оптимизацией интенсивности выпуска для максимизации прибыли при ограничениях времени ожидания пассажиров.

## Milestones

- [x] **Core DES Engine** — Реализация дискретно-событийного движка с очередью событий, обработка событий прибытия пассажиров, выпуска и прибытия автобусов
- [x] **Passenger Flow Modeling** — Пуассоновский поток пассажиров на остановках, усечённое нормальное распределение интервалов выпуска автобусов
- [x] **Metrics Collection** — Сбор метрик: время ожидания, длительность поездки, загрузка автобусов, количество обслуженных/потерянных пассажиров
- [x] **Optimization Algorithm** — Автоматическая оптимизация интенсивности выпуска автобусов с перебором значений и многократными прогонами для статистической достоверности
- [x] **Visualization System** — Генерация графиков и визуализаций результатов оптимизации (Matplotlib)
- [x] **Configuration Management** — Поддержка JSON-конфигурации с валидацией через JSONSchema, множество примеров конфигов
- [x] **Console CLI Interface** — Точка входа main.py с аргументами командной строки для запуска симуляции и оптимизации
